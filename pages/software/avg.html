How to Use

1. Running Scans

  1. Open AVG AntiVirus Free
  2. Click "Run Smart Scan" for a quick check
  3. For more options, click Computer in the sidebar
  4. Choose from Deep Scan, USB/DVD Scan, or Custom Scan
  5. Review results and quarantine threats

2. Managing Protection Shields

  1. Go to Menu > Settings > Basic Protection
  2. Toggle File Shield for real-time file scanning
  3. Toggle Web Shield for internet protection
  4. Toggle Email Shield for email scanning

3. Using File Shredder

  1. Go to Menu > More Tools > File Shredder
  2. Drag files or folders to shred
  3. Click "Shred All" to permanently delete
  4. Files cannot be recovered after shredding

4. Scheduling Scans

  1. Open AVG and go to Computer
  2. Click the gear icon next to Deep Scan
  3. Enable scheduled scans
  4. Set frequency, day, and time

Common Problems & Fixes

Problem 1: AVG not updating

  • Check internet connection
  • Go to Menu > Settings > General > Update
  • Click "Update Now" manually
  • Restart computer and try again

Problem 2: High CPU usage during scans

  • Schedule scans for idle times
  • Lower scan priority in settings
  • Enable Do Not Disturb mode during work

Problem 3: Legitimate files being blocked

  • Go to Menu > Settings > Exceptions
  • Add file path or application to whitelist
  • Check Quarantine to restore false positives

Problem 4: AVG conflicts with other software

  • Uninstall other antivirus programs first
  • Only run one antivirus for real-time protection
  • Add conflicting software to exceptions
