Introduction

Ruby is a dynamic, open-source programming language created by Yukihiro "Matz" Matsumoto in Japan. First released in 1995, Ruby was designed with an emphasis on programmer happiness and productivity, following the principle of least astonishment where the language behaves in a way that minimizes confusion for experienced users.

Ruby gained massive popularity with the release of Ruby on Rails in 2005, a web application framework that revolutionized web development. Today, Ruby powers millions of websites including GitHub, Shopify, Airbnb, and Basecamp, and remains one of the most beloved programming languages in the developer community.

Known for its elegant syntax that is natural to read and easy to write, Ruby is an excellent choice for beginners learning to program as well as experienced developers building complex applications. Its vibrant ecosystem of gems (libraries) and welcoming community make it a joy to work with.

Key Features

  • Elegant Syntax: Clean, readable code that feels natural and prioritizes developer happiness over unnecessary complexity
  • Pure Object-Oriented: Everything in Ruby is an object, including primitives, enabling consistent and intuitive code patterns
  • Dynamic Typing: Flexible type system that allows rapid development without verbose type declarations
  • Rich Standard Library: Comprehensive built-in libraries for file handling, networking, text processing, and more
  • RubyGems Package Manager: Access over 170,000 gems (libraries) to extend functionality and accelerate development
  • Metaprogramming: Powerful capabilities to write code that writes code, enabling domain-specific languages and flexible frameworks

Download & Installation

Official Download Link: https://www.ruby-lang.org/en/downloads/

Windows Installation (RubyInstaller)

  1. Visit rubyinstaller.org
  2. Download the latest Ruby+Devkit version (recommended)
  3. Run the installer and check "Add Ruby executables to your PATH"
  4. Complete the installation wizard
  5. When prompted, run the MSYS2 installation (press Enter)
  6. Open a new Command Prompt and verify: ruby --version
Tip: The Ruby+Devkit version includes MSYS2 for compiling native gem extensions.

macOS Installation

Method 1: Using Homebrew (Recommended)

# Install Homebrew if not installed /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)" # Install Ruby brew install ruby # Add to PATH (add to ~/.zshrc or ~/.bash_profile) echo 'export PATH="/opt/homebrew/opt/ruby/bin:$PATH"' >> ~/.zshrc source ~/.zshrc # Verify installation ruby --version

Method 2: Using rbenv

# Install rbenv brew install rbenv ruby-build # Initialize rbenv rbenv init # Install Ruby version rbenv install 3.3.0 rbenv global 3.3.0 # Verify ruby --version

Linux Installation

Ubuntu/Debian:

# Install dependencies sudo apt update sudo apt install git curl autoconf bison build-essential libssl-dev libyaml-dev libreadline6-dev zlib1g-dev libncurses5-dev libffi-dev libgdbm6 libgdbm-dev libdb-dev # Install rbenv curl -fsSL https://github.com/rbenv/rbenv-installer/raw/HEAD/bin/rbenv-installer | bash # Add to shell echo 'export PATH="$HOME/.rbenv/bin:$PATH"' >> ~/.bashrc echo 'eval "$(rbenv init -)"' >> ~/.bashrc source ~/.bashrc # Install Ruby rbenv install 3.3.0 rbenv global 3.3.0

Fedora/RHEL:

sudo dnf install ruby ruby-devel

Arch Linux:

sudo pacman -S ruby

How to Use

1. Running Ruby Code

Interactive Ruby (IRB):

# Start interactive Ruby shell irb # Try some Ruby code >> puts "Hello, World!" Hello, World! >> 2 + 2 => 4 >> exit

Running a Ruby Script:

# Create a file called hello.rb puts "Hello, World!" # Run it ruby hello.rb

2. Basic Ruby Syntax

# Variables name = "Ruby" age = 29 # String interpolation puts "#{name} is #{age} years old" # Arrays fruits = ["apple", "banana", "orange"] fruits.each { |fruit| puts fruit } # Hashes person = { name: "John", age: 30 } puts person[:name] # Methods def greet(name) "Hello, #{name}!" end puts greet("World") # Classes class Dog def initialize(name) @name = name end def bark "#{@name} says Woof!" end end dog = Dog.new("Buddy") puts dog.bark

3. Using RubyGems

# Install a gem gem install bundler gem install rails gem install sinatra # List installed gems gem list # Update gems gem update

4. Creating a Gemfile with Bundler

# Create a new directory mkdir my_project cd my_project # Initialize Gemfile bundle init # Edit Gemfile and add gems gem 'sinatra' gem 'puma' # Install dependencies bundle install

Common Problems & Fixes

Problem 1: ruby command not found

Solution:

  • Verify Ruby is installed: check installation directory
  • Add Ruby to your PATH environment variable
  • Windows: Reinstall and check "Add to PATH" option
  • Restart your terminal after installation

Problem 2: Gem installation fails with permission error

Solution:

  • Use a version manager like rbenv or rvm (recommended)
  • Install gems to user directory: gem install --user-install gemname
  • Avoid using sudo for gem installations

Problem 3: Native extension build fails

Solution:

  • Windows: Install Ruby+Devkit version with MSYS2
  • macOS: Install Xcode Command Line Tools: xcode-select --install
  • Linux: Install build-essential and development headers
  • Ensure OpenSSL development packages are installed

Problem 4: SSL certificate errors

Solution:

  • Update RubyGems: gem update --system
  • Windows: Update certificate bundle
  • Set SSL_CERT_FILE environment variable if needed

Problem 5: Bundle install fails

Solution:

  • Update Bundler: gem install bundler
  • Clear bundle cache: bundle clean --force
  • Delete Gemfile.lock and try again
  • Check for conflicting gem versions

Problem 6: Wrong Ruby version being used

Solution:

  • Check which Ruby: which ruby or where ruby
  • Use rbenv: rbenv versions and rbenv global x.x.x
  • Create .ruby-version file in your project
  • Ensure version manager is properly initialized in shell config

Whether you're building a Ruby on Rails application, writing automation scripts, or learning programming for the first time, Ruby provides an elegant and productive experience. The supportive community and wealth of resources make it easy to get help and continue learning.
