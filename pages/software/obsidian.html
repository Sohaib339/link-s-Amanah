Introduction

Obsidian is a powerful and extensible knowledge base application that works on top of your local folder of plain text Markdown files. Unlike cloud-based note-taking apps, Obsidian gives you full ownership and control of your data.

Built around the concept of linked thinking, Obsidian lets you create connections between your notes using wiki-style links. The result is a personal knowledge graph that helps you discover relationships between ideas and build a "second brain."

Obsidian is free for personal use and offers optional paid services for syncing and publishing. Its vibrant plugin ecosystem allows you to customize and extend its functionality to match your exact needs.

Key Features

  • Local-First: Your notes are plain Markdown files stored on your device
  • Backlinks: See all notes that link to the current note
  • Graph View: Visualize the connections between your notes
  • Plugins: 1000+ community plugins for endless customization
  • Themes: Customize appearance with community themes
  • Canvas: Infinite canvas for visual thinking and planning
  • Templates: Create reusable templates for consistent note-taking
  • Tags & Folders: Organize notes with flexible hierarchies
  • Search: Powerful full-text and property-based search
  • Sync (Paid): End-to-end encrypted sync across devices
  • Publish (Paid): Publish notes as a website

Download & Install on Windows

Official Download: https://obsidian.md/download
  1. Visit obsidian.md/download
  2. Click "Download for Windows"
  3. Run the downloaded Obsidian installer (.exe)
  4. Follow the installation prompts
  5. Launch Obsidian
  6. Create a new vault or open an existing folder
  7. Start creating and linking your notes!
Tip: A "vault" is simply a folder containing your Markdown files. You can create multiple vaults for different projects.

Download & Install on macOS

Official Download: https://obsidian.md/download
  1. Download the macOS .dmg file from obsidian.md
  2. Open the downloaded .dmg file
  3. Drag Obsidian.app to your Applications folder
  4. Launch Obsidian from Applications
  5. Create a new vault or open an existing folder

Alternative: Using Homebrew

brew install --cask obsidian

Download & Install on Linux

Official Download: https://obsidian.md/download

AppImage (Recommended)

  1. Download the .AppImage file from obsidian.md
  2. Make it executable and run:
chmod +x Obsidian-*.AppImage ./Obsidian-*.AppImage

Flatpak

flatpak install flathub md.obsidian.Obsidian

Snap

sudo snap install obsidian --classic

Debian/Ubuntu (.deb)

sudo dpkg -i obsidian_*.deb
Tip: Flatpak and Snap versions update automatically and are sandboxed.

Conclusion

Obsidian stands out in the crowded note-taking space by putting you in complete control of your data. Your notes are stored as plain Markdown files that you own forever—no vendor lock-in, no subscription required for basic use.

Whether you're building a personal knowledge base, writing a book, managing research, or just taking daily notes, Obsidian's flexibility and powerful linking features help you think better and work smarter.
